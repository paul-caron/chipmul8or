
# Chipmul8or

Plays Chip8 ROM files.

Needs ncurses library installed. Sound is working through the terminal bell character. It's just wonderful.

# Compile

``` sh
git clone https://www.github.com/paul-caron/chipmul8or
cd chipmulator
make
```

# Run

./main "Name of ROM file"

eg:
``` sh
./main PONG
```
